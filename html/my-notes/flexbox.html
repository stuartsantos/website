Flexbox

Flexbox is an incredible CSS property that makes web design and layout so much simpler to work with. However, since it's a relatively new addition to the CSS tool set, you can find that certain browsers and devices act a bit strange when you start using it. I added flexbox to the CSS we use for all of our AIG Consumer websites to help with layout and positioning of elements on the page irrespective of DOM order.

There are two key things to keep in mind when using the flexbox propery. First of all, you want to remove all floats that may be present in your initial CSS. When you set a flex container and the child elements are floated, you get some really strange behavior or it doesn't do anything at all.

The second important issue to keep in mind is if you have to support the Blackberry 7 browser. For whatever reason, flexbox is supported, however it does not allow the use of row wrapping. Instead, you should be declaring the orientation and direction of your flex container to be vertical and column.

Once you have all that set in place, it should be relatively easy to style the layout of your page. You can see below the code we finally used to make sure the flex ordering and wrapping worked propely. Unfortunately, the CMS we use at AIG auto-ouputs the center grid elements after the right grid elements, so we needed a way to reorder the DOM elements for mobile so that the less important right grid information came later in the page.

#grid_wrapper { display: -webkit-box; display: -webkit-flex; display: -moz-box; display: -ms-flexbox; display: flex; -webkit-box-orient: vertical; -moz-box-orient: vertical; -webkit-box-direction: normal; -moz-box-direction: normal; -webkit-flex-direction: column; -ms-flex-direction: column; flex-direction: column; width: 100%; } #gridRight { -webkit-box-ordinal-group: 2; -moz-box-ordinal-group: 2; -ms-flex-order: 2; -webkit-order: 2; order: 2; width: 100%; padding-left: 0; float: none; } #gridCenter { -webkit-box-ordinal-group: 1; -moz-box-ordinal-group: 1; -ms-flex-order: 1; -webkit-order: 1; order: 1; }
